Usage

1. Import the component

import {{ '{' }} ButtonComponent {{ '}' }} from '@angular-bits/ui';

2. Add to your component imports

imports: [ButtonComponent]

3. Use in your template

<ui-button status="primary" size="medium" appearance="filled">Button Text</ui-button>
